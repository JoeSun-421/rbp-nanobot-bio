3. **验收命令固化进 README：** `doctor` → `accept-golden` → `accept-llm` → `eval-plan`（本机产物路径）。  
4. **措辞：** README / 对外材料统一 C1（raw `p_hat`；非校准 P(bind)）。  
5. **评测证据：** 补齐 heavy LOO / ECE / 消融报告路径后再写入里程碑完成声明。  
6. **文档：** 保持 `proposal*` / `delivery要求*` / 本清单在 `docs/`；重要验收说明同步进仓根 README（可提交）。
