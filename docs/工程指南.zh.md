## 9. 改动门禁

### MUST NOT

- 编辑 `rhobind_agent_delivery/`（registry、weights、脚本）。科学仅经 App bridge。
- 对未见靶点无 proxy donors 时调用 \(f_\theta\) / `predict_interaction`（提案 §2–§4）。目录内 own-head 仅 Stage 0 / near-known。
- 用 LLM 编造 `p_hat` / `prob`。OOM/timeout 时输出 `p_hat=null`。
- 将结构 / AF3 失败当作相似度 `0`。回退序列/结构域，并保持低置信度。
- 无评测证据且未同步 `config/defaults.yaml` 与 `tests/test_proposal_compliance.py` 时，改动 Table 3 默认（`n_cand=5`、`tau_drop=0.30`、near-match ≥95%、标签切分 0.75/0.50/0.25、nanobot 框架）。
- 在线写权重或自动改 delivery registry。自演进仅离线（提案 §7）。
- 无门禁 + nested-split 证据（`delta_auprc > 0` 或 HOLD）时 promote evolved 配置。
- 增加第三套 tools 树或复活 `rbp_agent/_proposal_sot`。只编辑 `nanobot/` SoT；workspace skill 为 sync 符号链接/副本。
- 在无真实 `RNA_FM_CHECKPOINT` 时给 mock RNA-FM 融合权重（`rna_embed` / `rna_fm` 保持 0）。
- 用提案/聊天笔记替换产品 `README.md`；工程门禁放本节。
- 提交 API keys、`.env`、或 `~/.nanobot/config.json`。
- 将科学 torch（RhoBind/ESM）导入 nanobot 进程；保持 conda/subprocess 隔离。
- 引入 LangGraph / CrewAI / AutoGen 作为产品依赖。

### MUST

- 保持三层：Agent Controller（nanobot）/ Toolkit / Predictor + 离线 Validation Evaluator（提案 §3）。
- 产品路径：`Nanobot.from_config` → `run` / `run_streamed`（`nanobot-bio agent|chat`）。MVP/评测优先 `ephemeral=True`。
- 保留 Stage 0→1→2→3 语义与**两个 LLM 检查点**。允许确定性 `fuse_*` 基线。
- Stage 0：严格 UniProt 匹配 → own-head 一次后 STOP；identity ≥95% → near-known Fast Path 并在 explanation 标注。
- 上限 \(N_{\mathrm{cand}}\le5\)；丢弃融合相似度 \(<0.30\)。
- 工具契约：JSON Schema、error envelope、`latency_ms`；retrieve 类 `read_only=True`。
- Stage-3：checklist 失败 ≥2（或等价 evidence flags）时，`normalize_verdict` 强制 `confidence=low`。
- Skill SoT：`nanobot/skills/rbp-agent/SKILL.md`。科学主张须附 `eval-plan` / `evolve-eval` 报告路径。
- 模型元数据在 `config/defaults.yaml` → `models:`（doctor 能力矩阵）。

### 实现口径（相对提案原文的合法偏离）

- **结构：** AFDB → Foldseek 为主；无 AFDB 时默认 AF3 回退（`axes.use_af3=true` / `use_af3_fallback=true`）；探针失败 → caveat，≠ sim 0。
- **序列：** ESM-C + MMseqs 双轴（`hits_emb` / `hits_seq`）；蛋白侧不用 BLASTn。
- **聚合：** 默认 `predict.aggregate: weighted`（§4 \(\sum s\cdot p\cdot c\)）；权威 `s_i` 来自 `commit_proxy_candidates`。
- **`p_hat`：** predictor **raw**（加权后）；`confidence` 为规则 + checklist。不宣称已校准 P(bind)。`score_calibration` 为 Delivery v2 OUT。
- **融合：** 确定性 `fuse_similarity_views` 为证据；LLM Checkpoint 1 标定并 `commit_proxy_candidates`。
