# `rbp_agent.backends`

**English** | [中文](README.zh.md)

Backend adapters for **nanobot-bio**. The primary adapter is [`delivery/`](delivery/README.md) (read-only science bridge). Additional backends (e.g. [`rna_fm/`](rna_fm/) for agent-local RNA embedding similarity) live beside delivery and must not mutate delivery sources.
